# 🏛️ Atenas IA

> Inteligência Operacional para Centrais de Atendimento

## 📌 Sobre o Projeto

Atenas IA é uma plataforma inteligente desenvolvida para automatizar a análise de KPIs de operações de atendimento, principalmente em centrais bancárias e contact centers.

O sistema recebe um arquivo Excel, processa os indicadores utilizando um algoritmo desenvolvido em **R**, gera um novo arquivo tratado e utiliza Inteligência Artificial para produzir análises detalhadas, responder perguntas e auxiliar supervisores, coordenadores e gerentes na tomada de decisão.
